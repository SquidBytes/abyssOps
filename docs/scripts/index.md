# Scripts

Reusable scripts for common operational tasks. Each script is self-contained with usage documented inline (`-h`) or in the corresponding doc page.

Raw script files live in [`scripts/`](https://gitlab.com/NAMESPACE/abyssOps/-/tree/main/scripts) in the repo root.

## Available Scripts

| Script | Description | Docs |
|--------|-------------|------|
| `vscode/install-vscode-server.sh` | Install VS Code Server offline — local, SSH, or Docker | [Remote SSH Offline](../vscode/remote-ssh-offline.md) |
